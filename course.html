SYLLABUS

HTML

HTML is the framework that defines the structure and layout. It tells your browser what content to show, like text, images, and links. So, it's basically the language that web browsers understand to display websites.

CSS

CSS adds style and design. It decides how things look—colors, fonts, spacing, and all the visual stuff. So, if HTML is the blueprint, CSS is the paint, furniture, and decorations that make your website look good and feel inviting.

JAVA SCRIPT

It's a scripting language that brings your websites to life. Think of it as the puppeteer behind the scenes—making things move, pop up, and change color on your screen. It adds interactivity, making web pages more dynamic and engaging. So, when you click a button, see a cool animation, or a form gives you feedback without reloading the page, that's the magic of JavaScript!

JQUERY

It simplifies common things like handling events, animations, and AJAX requests. So, instead of writing a bunch of code, you can use jQuery to get things done faster.

JDBC

JDBC or Java Database Connectivity, is like a translator for Java and databases. When your Java program needs to talk to a database, JDBC steps in. It helps Java understand what the database is saying and vice versa. It's the bridge that lets your Java code ask the database questions, fetch data, or update information.
